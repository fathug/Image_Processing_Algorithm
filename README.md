# Image_Processing_Algorithm

基于C#实现的图像处理算子

| 文件夹           | 包含算子                                        | 功能                         |
| ---------------- | ----------------------------------------------- | ---------------------------- |
| Line_Fitting     | 灰度转换<br>梯度计算<br>坐标提取<br>LSM直线拟合 | 从边缘图到直线               |
| TemplateMatching | 模板匹配                                        | 找到图像中与模板最相似的区域 |
